# Don't Pay for Eightify

An open-source YouTube video summarizer that turns learning content into concise bullet points.

## Philosophy
"Don't pay for what you can build."
This project replicates the core functionality of paid services like Eightify, creating high-quality summaries from YouTube video transcripts using LLMs.

## Features
- Extract transcripts from YouTube videos.
- Synthesize transcripts into structured bullet points.
- Focus on learning/educational content.
